About:
   Tune list is cross platform mobile app for recording keyboard-based music snippets and adding them to in-app playlist

Steps:
1. Download Node.js and npm
   Node.js: https://nodejs.org
   npm install -g expo-cli

2. Download Expo Go app or Android/iOS simulator

3. Clone Repository and enter
   git clone https://github.com/aaronnewmanj/Tune-List.git
   cd Tune-List
   cd TuneList
   cd TuneList (nested folder)

4. Install Dependancies and initialize
   npm install
   npx expo start

5. Run Application
   Scan QR code in terminal for access to app
